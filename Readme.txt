wc.c is a C Program to count the number of words, characters and lines in a file.
The single process approach cannot scale to handle large files. multiwc.c is a C Program to count the number of words, characters and lines in a file using multiple child processes. This program receives the number of child processes and an input file name
through the command-line argument.
multiwc_crash introduces crash handling to the multiprocessor word counter.
